# Machine Learning

## 1. K-Means Clustering

> Partitions a dataset into K distinct clusters by iteratively assigning points to the nearest centroid.

Use Case: Market segmentation and identifying customer groups.

Time Complexity: O(n · k · i) where i is iterations.

## 2. PageRank

> Assigns numerical weights to hyperlinked documents by measuring the quantity and quality of links pointing to them.

Use Case: Search engine algorithms ranking web pages.

Time Complexity: O(V + E) per iteration.
